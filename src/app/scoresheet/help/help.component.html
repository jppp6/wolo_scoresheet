Wolo Gamesheet Help & Information

Goals:
Input is: player number, team color, goal and time.
Player number options: 1 to 20
Team color options: l, light, w, white, h, home, d, dark, b, blue, a, away
Goal options: g, goal
Examples: 6bg100, 4whiteg10, 10darkgoal700, 14lgoal123

Next Quarter:
Input is: x (it can be repeated)
Examples: x, Xx, XXXXXX

Timeouts:
Input is: timeout, team color and time.
Timeout options: t, to, timeout, time
Team color options: l, light, w, white, h, home, d, dark, b, blue, a, away
Examples: tb100, timeoutwhite123, todark700, timew111

Cards:
Input is: person, team color, type of card and time.
Person options: 1-20, hc, ac1, ac2
Team color options: l, light, w, white, h, home, d, dark, b, blue, a, away
Type of card options: r, red, y, yellow.
Examples: 6lightred100, acdarkred111, aabluey606, hcwhitered111

Cap Swap:
Input is: player1, swap, player2, team color and time.
Player options: 1-20
Swap options: s, swap
Team color options: l, light, w, white, h, home, d, dark, b, blue, a, away
Examples: 6swap9b100, 2s10blue111

Exclusion:
Input is: player, team color, exclusion and time.
Player options: 1-20
Team color options: l, light, w, white, h, home, d, dark, b, blue, a, away
Exclusions options: e, exclusion, k, kick, kickout
Examples: 15blueexclusion111, 6lightkickout120, 12be600

Brutality:
Input is: player, team color, brutality and time.
Player options: 1-20
Team color options: l, light, w, white, h, home, d, dark, b, blue, a, away
Brutality options: b, brutal, brutality
Examples: 6bluebrutal111, 7whiteb500, 10bb10

Wolo Scoresheet Help & Information

Goal, Exclusion, Brutality Format
Input Format: <player><team><event><time>
Player options: 1-20
Team options: l/h/d/a (light/home/dark/away)
Event options:
-g: goal
-e: exclusion
-b: brutality
Time options: 0 to 999 (0:00 to 9:99)

Next Quarter Format
Input Format: q (quarter)

Timeout Format
Input Format: <timeout><team><time>
Timeout options: t (timeout)
Team options: l/h/d/a (light/home/dark/away)
Time options: 0 to 999 (0:00 to 9:99)

Card Fromat
Input Format:<person><team><card><time>
Person options: 1-20/c/a1/a2 (coach, assistant1, assistant2)
Team options: l/h/d/a (light/home/dark/away)
Card options: r/y (red,yellow)

Cap Swap Format
Input Format: <player1><swap><player1><team><time>
Input is: player1, swap, player2, team color and time.
Player1 options: 1-20
Swap options: s (swap)
Player2 options: 1-20
Team options: l/h/d/a (light/home/dark/away)
Time options: 0 to 999 (0:00 to 9:99)
